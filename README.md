🚀 Curso de Bash y Terminal Linux - MoureDev

En este repositorio comparto mi progreso, apuntes y scripts desarrollados a lo largo del Curso de Bash y la Terminal desde Cero impartido por Brais Moure (MoureDev) en su canal de YouTube.

El objetivo de este proyecto es dominar la terminal de Linux como mi entorno nativo de trabajo , aprender a interactuar con el sistema de forma eficiente y sentar las bases de automatización necesarias para la infraestructura y la ciberseguridad.
🛠️ Tecnologías y Conceptos Aprendidos

    Navegación y Gestión de Archivos: Desplazamiento por el sistema de archivos, manipulación de directorios y uso de comandos esenciales para listar o localizar elementos de forma precisa en el disco.

    Filtrado y Procesamiento de Texto: Uso de herramientas de filtrado como grep y awk combinadas con expresiones regulares para auditar diccionarios, estructurar datos o parsear archivos de texto masivos.

    Automatización con Scripts (.sh): Creación de scripts automatizados ejecutables mediante el uso del Shebang (#!), el control de permisos del sistema operativo y la elevación segura de privilegios administrados.
